1. load createDW.sql file in your mysql, check if the tables and schema is created.
2. load the project in your eclipse IDE, and add your mysql connector jars. remove the existing one.
3. open NRD class, and setup the run configrations. and run the application
4. enter your root name, password.
5. enter your source schema name, and destination schema name.
6. all data will be loaded.
